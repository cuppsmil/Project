# README

## Описание

В этом проекте собраны решения для 3 учебных кейсов, выполненных в **Jupyter Notebook в Google Colab**.  
Все задачи реализованы на **Python** с упором на автоматизацию рутинных процессов обработки данных и расчета метрик.

Проект включает:

1. **Объединение CSV-файлов с продажами магазинов**
2. **Формирование отчета по задолженностям студентов**
3. **Расчет продуктовых метрик по активности пользователей**
